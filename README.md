# トミカ コレクション カタログ (Tomica Collection Catalog)

子どものトミカを写真とメモで記録・管理するためのローカルWebアプリケーションです。

サーバーへのデータ送信は一切行わず、お使いのブラウザ内（IndexedDB）にデータを安全に保存します。個人情報を送信することなく、すべての機能を完全無料でご利用いただけます。

---

## 🌟 主な機能

1. **トミカの記録・編集**
   - 名前、トミカ番号、メーカー/車種、カテゴリ、カラー、状態、箱の有無、写真、メモを記録できます。
2. **写真アップロード＆カメラ撮影**
   - スマホのカメラでトミカを撮影してその場で登録したり、アルバムから写真を追加できます。
3. **高機能な検索・絞り込み**
   - キーワード検索（名前、番号、メーカー、メモから検索）
   - カテゴリフィルター（定番トミカ、トミカプレミアム、ロングタイプ、ドリームトミカなど）
   - 並び替え（登録が新しい順、古い順、トミカ番号順、50音順）
4. **ダッシュボード統計**
   - トミカの総数、お気に入り数、箱あり数、きれいな状態の数をビジュアル表示。
5. **JSONバックアップ（インポート/エクスポート）**
   - 登録したデータを写真データも含めて丸ごと1つのJSONファイルとしてダウンロード（エクスポート）できます。別の端末やブラウザでデータを読み込む（インポート）ことで、データの移行やバックアップが簡単に行えます。
6. **ダークモード/ライトモード**
   - プレミアム感のあるシックなダークモードと、明るく見やすいライトモードをワンタップで切り替えられます。

## 🚀 起動方法

このアプリケーションはローカル環境で動作します。以下のいずれかの方法で起動できます。

### 方法 1: ダブルクリックで起動（最も簡単）
1. フォルダ内の `index.html` をダブルクリックしてブラウザで開きます。
2. そのまま使用を開始できます。
   *(※ブラウザのセキュリティ設定によっては、`file://`プロトコルでIndexedDBが制限される場合があります。その場合は方法2をお試しください)*

### 方法 2: ローカルサーバーを起動（推奨・動作が安定）
本ディレクトリでローカルサーバーを起動することで、すべての機能が最も安定して動作します。

#### A) Node.js をお使いの場合
Node.js環境がある場合は、以下のコマンドで簡単に起動できます。

```bash
# ローカルサーバーの起動 (本ディレクトリで実行)
npx serve
```
または、`npm` のスクリプトを利用します：
```bash
npm run start
```
起動後、ブラウザで `http://localhost:3000` (または表示されたアドレス) にアクセスします。

#### B) Python をお使いの場合 (Mac/Linux標準搭載)
ターミナルを開き、本ディレクトリで以下のコマンドを実行します。

```bash
python3 -m http.server 8000
```
起動後、ブラウザで `http://localhost:8000` にアクセスします。

## 📂 ファイル構成

- `index.html` : サイトの骨組みとダイアログのマークアップ
- `index.css`  : プレミアム感のあるガラスモルフィズムやダークモード対応のスタイルシート
- `app.js`     : IndexedDB制御、データのCRUD処理、検索・フィルタ、UI構築のロジック
- `package.json` : ローカルサーバー起動用の設定ファイル
